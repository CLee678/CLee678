Knowledge Acquisition

In the Web Mapping and GIS course, I have acquired a diverse range of skills and knowledge in geospatial data visualization through various mapping assignments. I have learned how to utilize GeoJSON data and JavaScript libraries like Leaflet and Google Charts to create interactive maps and charts, incorporating features such as markers, tooltips, and click events to enhance user engagement and data presentation. Additionally, I have gained knowledge in accessing external API keys such as the ESRI Software Development Kit and Mapbox, and experience in implementing advanced functionalities like layer controls and data filtering. These experiences have not only expanded my technical capabilities but also deepened my understanding of spatial data analysis and visualization techniques, equipping me with valuable skills applicable across various projects.

Map with US Cities

This is a basic interactive map featuring three US cities: Philadelphia, New York, and Washington DC. Through this assignment, I have learned how to utilize the JavaScript library Leaflet to create interactive maps and create markers with bind popup method.

Population of Harrisburg and Lancaster

This interactive map displays proportional circles representing the populations of Harrisburg and Lancaster cities. Throughout this assignment, I have learned to utilize the 'L.circle' and tooltip methods, as well as how to format text to be bold and italicized, and how to change the basemap.

Mapping Frank Glavin Playground

This map shows only the Frank Glavin Playground. Throughout this assignment, I have learned how to utilize GeoJSON data, create access tokens for Mapbox basemaps, and implement 'pointToLayer' function to define how GeoJSON points spawn Leaflet layers, and filter functions to include the required features.

US Electric Power Generation

This is a proportional symbol interactive map showing the amount of electric power generated in the US, including power generated from Biomass, Geothermal, and Solar sources. Throughout this assignment, I have learned how to provide users with a layer control to toggle the visibility of various power plant data layers on and off, and how to style the label text within the layer control to correspond with the color used to encode a specific power fuel source.

Art Galleries in New York

This map displays all the art galleries located in New York City. By clicking on a point representing each gallery, users can view details such as the gallery's name, city location, and telephone number. Through this assignment, I have gained knowledge about accessing data from the ArcGIS REST Services Directory and creating maps with JavaScript using the ESRI SDK.

California Block Group - 2018 Population Counts

This choropleth map illustrates the 2018 population distribution across block groups in California. Through this assignment, I have acquired skills in effectively displaying data using quantitative categories and selecting appropriate colors and stops for visualization. Additionally, I implemented a clickable popup feature that identifies the geographic unit and provides users with informative messages regarding the population value of the selected enumeration unit.

World Earthquake Distribution

This map displays the distribution of earthquakes worldwide. It visualizes earthquake magnitudes using proportional symbols, where larger symbols represent higher magnitudes. Clicking on a symbol reveals detailed information about the earthquake's location and magnitude.

Interactive Charts

The line chart and bar chart feature interactive elements. I have gained experience in JavaScript coding to create Google Charts for visualizing and presenting data. This process has taught me how to implement interactive features like tooltips and click events, as well as customize charts to better visualize the data.

Find My Hidden Location!

This map features a hidden location marker! Feel free to explore it by interacting with the map. The marker will become visible when you click within a 100-mile radius of the hidden location. Through this mapping assignment, I have gained knowledge in categorizing popup information based on proximity to the hidden marker using 'if' and 'else if' statements in my code. Additionally, I have learned how to use the 'clearLayers' method to remove previous marker clicks and how to associate icons with markers effectively.
